SQL, Structure Query Lenguague (lenguaje de consulta estructurado) es un lenguaje de programacion para trabajar con base de datos relacionales como MySQL, Oracle, etc. MySQL es un interpretador de SQL es un servidor de base de datos.
My SQL permite crear base de datos y tablas, insertar datos, modificarlos, eliminarlos, ordenarlos, hacer consultas y realizar operaciones, etc, resumiendo: administrar base de datos.

Este tutorial tiene por objetivo acercar los conceptos iniciales para introducirse en el mundo de las bases de datos.

Tipos de datos en MySQL

varchar

Se usa para almacenar cadenas de caracteres. Una cadena es una secuencia de caracteres. Se coloca entre comillas (simple): 'Hola'
El tipo "varchar" define una cadena de longitud variable en la cual determinamos el maximo de caracteres. Puede guardra hasta 255 caracteres. Puede almacenar cadenas de hasta 30 caracteres, definimos un campo de tipo varchar.(30)

int

Se usa para guardar valores numericos enteros, de -2000000000 a 2000000000 aproximadamente. definimos campos de este tipo cuando queremos representar como por ejemplo, cantidades.

Tipos de datos en MySQL

TEXTO: Para almacenar texto usamos cadena de caracteres. Las cadenas se colocan entre comillas simple. Podemos almacenar digitos con los que no se realizan operaciones matematicas, por ejemplo, codigos de identificacion, numero de documentos, numeros telefonicos. Tenemos los siguientes tipos: varchar, char, y text.

NUMEROS: Existe variedad de tipos numericos para represenar enteros, negativos, decimales. Para almacenar valores enteros, por ejemplos en campos que hacen referencias a cantidades, precios , etc.,usamos el tipo integer. Para almacenar valores con decimales utilizamos: float o decimal.

FECHA Y HORAS: Para guardar fecha sy horas dispone de varios tipos: date(fecha), datetime(fecha y hora), time(hora), year (año)y times tamp.
